Thank you for downloading DesignOra WordPress theme!!

Installation:
1. Extract Designora-theme.zip and upload the theme folder via FTP to your wp-content/themes/ directory.
2. Go to your WordPress Admin Dashboard > Design > Themes and select the DesignOra Theme screenshot.
3. Activate Designora Theme.
